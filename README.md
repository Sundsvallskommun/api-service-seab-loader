# SeabLoader

## Beskrivning
SeabLoader är en tjänst som gör det möjligt att från InXchange ladda SEAB:s pdf-fakturor till InvoiceCache.

## Tekniska detaljer

### Integrationer
Tjänsten integrerar mot:

* Mikrotjänst InvoiceCache

## Leverantör

Sundsvalls kommun
